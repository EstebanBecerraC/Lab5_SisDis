|===========================|
|===|   Como ejecutar   |===|
|===========================|

Primero debe estar el Broker y el servidor Fulcrum funcionando y deben existir sectores dentro de la carpeta "Sectores" en donde se encuentra el servidor Fulcrum.

Ejecutando "make" se compila y ejecuta en un docker.
Con "make clean" se borra lo creado.
Esto considerando sistema linux.
